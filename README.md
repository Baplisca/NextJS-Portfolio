# NextJS-Portfolio

## Getting Started

```bash
git clone git@github.com:Baplisca/NextJS-Portfolio.git
cd NextJS-Portfolio
npm i
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
